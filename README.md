# 🧠 Decision Lab

## 📌 Sobre o projeto

O Decision Lab é um laboratório de análise de decisões sob risco e incerteza.

O objetivo é usar dados para entender como risco, retorno e comportamento humano influenciam resultados em diferentes contextos como investimentos, poker, imóveis e criptomoedas.

---

## 🧠 Visão do projeto

O Decision Lab não é apenas um projeto de análise de dados.

Ele é um laboratório para estudar como decisões humanas se comportam sob incerteza.

A ideia central é transformar experiências subjetivas (investimentos, apostas, escolhas financeiras) em dados estruturados, permitindo análise objetiva de risco e retorno.

---

## 🎯 Problema central

Como melhorar decisões quando o resultado é incerto e depende de variáveis imprevisíveis?

---

## 🧱 Estrutura do projeto

- Risco vs Retorno  
- Decisão sob Incerteza  
- Comportamento Humano  
- Índice de Liberdade Financeira  

---

## 📊 Dados

O projeto utiliza um dataset simulado contendo decisões em:

- Poker  
- Imóveis  
- Ações  
- Criptomoedas  

Cada decisão contém variáveis como:
- risco
- retorno esperado
- retorno real
- resultado
- contexto

---

## 🚧 Etapas do projeto

1. Construção da base de dados (SQL)
2. Simulação de decisões (dataset inicial)
3. Análise exploratória (EDA)
4. Comparação de risco vs retorno
5. Modelagem de padrões de decisão

---

## 🎯 Impacto esperado

Este projeto busca responder perguntas como:

- Decisões mais arriscadas geram maior retorno?
- Como o comportamento humano influencia perdas financeiras?
- Quais ativos possuem melhor relação risco/retorno?
- É possível medir qualidade de decisão ao longo do tempo?

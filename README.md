# Decision Lab

## 📌 Sobre o projeto

O Decision Lab é um laboratório de análise de decisões sob risco e incerteza.

O objetivo é usar dados para entender como risco, retorno e comportamento humano influenciam resultados.

---

## 🎯 Problema

Como melhorar decisões quando o resultado é incerto?

---

## 🧱 Estrutura do projeto

- Risco vs Retorno
- Decisão sob Incerteza
- Comportamento Humano
- Índice de Liberdade Financeira

---

## 🗂 Dados

Simulação de decisões envolvendo:
- Poker
- Imóveis
- Ações
- Criptoativos
